# 4EX - Современная Платформа Обмена Валют

Современный веб-сайт обмена криптовалют и электронных денег с интуитивным интерфейсом, темной темой и поддержкой множества направлений обмена.

![Tech Stack](https://img.shields.io/badge/React-18-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)
![Vite](https://img.shields.io/badge/Vite-6-purple)
![Tailwind CSS](https://img.shields.io/badge/TailwindCSS-3-cyan)

## 🚀 Функциональность

### Реализовано

- ✅ **Главная страница** с hero-секцией и преимуществами сервиса
- ✅ **Калькулятор обмена** с автоматическим расчетом курсов
- ✅ **Темная/светлая тема** с сохранением в LocalStorage
- ✅ **Адаптивный дизайн** для мобильных и десктопных устройств
- ✅ **Поддержка валют**:
  - Криптовалюты (BTC, ETH, USDT TRC20/ERC20, USDC)
  - Электронные кошельки (Payeer, Perfect Money, AdvCash)
  - Банковские карты (Visa/MC в RUB, USD, EUR)
  - Наличные (Москва, СПБ)
- ✅ **Управление состоянием** через Zustand
- ✅ **Маршрутизация** с React Router
- ✅ **TypeScript** для типобезопасности
- ✅ **Валидация форм** и обработка ошибок
- ✅ **Mock данные** для демонстрации

## 🚀 Быстрый старт

### Требования

- Node.js 18+ 
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd exchange
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите dev-сервер:
```bash
npm run dev
```

4. Откройте http://localhost:5173 в браузере

### Доступные скрипты

```bash
npm run dev          # Запуск dev-сервера
npm run build        # Production сборка
npm run preview      # Просмотр production сборки
npm run lint         # Проверка кода (если настроен ESLint)
```

## 💡 Основные возможности

### Калькулятор обмена

- Выбор валюты "отдаю" и "получаю"
- Автоматический расчет с учетом курса и комиссии
- Отображение резервов валют
- Валидация минимальных и максимальных сумм
- Кнопка swap для обмена направлений

### Темизация

- Темная тема по умолчанию
- Переключение через кнопку в header
- Сохранение предпочтений в LocalStorage
- Автоматическое определение системной темы

### Валюты

Поддерживаются 4 типа валют:

1. **Криптовалюты**: BTC, ETH, USDT (TRC20, ERC20), USDC
2. **Электронные кошельки**: Payeer, Perfect Money, AdvCash
3. **Банковские карты**: Visa/MC (RUB, USD, EUR)
4. **Наличные**: по городам России

## 🎨 Дизайн

### Цветовая палитра

**Темная тема (по умолчанию):**
- Фон: `#0f172a`, `#1e293b`
- Текст: `#f1f5f9`, `#cbd5e1`
- Акцент: градиент `#3b82f6` → `#8b5cf6`

**Светлая тема:**
- Фон: `#ffffff`, `#f8fafc`
- Текст: `#0f172a`, `#334155`
- Акцент: тот же градиент

### Responsive breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

## 📝 Форматирование

- **Числа**: разделитель тысяч - пробел, десятичный - запятая (1 000,50)
- **Даты**: ДД.ММ.ГГГГ (15.01.2024)
- **Время**: ЧЧ:ММ (14:30)
- **Криптовалюты**: до 8 знаков после запятой
- **Фиатные валюты**: 2 знака после запятой

## 🔐 Валидация

Реализована валидация для:

- Email (regex)
- Telegram username (@username)
- Bitcoin адреса
- Ethereum адреса
- Номера банковских карт (алгоритм Луна)
- Номера электронных кошельков
- Суммы с проверкой лимитов и резервов

## 🌍 Язык

Интерфейс на русском языке с подготовленной структурой для добавления мультиязычности.
