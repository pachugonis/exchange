# ExchangeKit — быстрая установка (шпаргалка)

Краткая версия. Подробности — в [ИНСТРУКЦИЯ_ПОКУПАТЕЛЯ.md](ИНСТРУКЦИЯ_ПОКУПАТЕЛЯ.md).

## Перед началом
- VPS с **Ubuntu 24.04**, доступ **root**.
- Домен с **A-записью** на IP сервера.
- На руках: файл `install.sh`, **лицензионный ключ** `LIC-...`, **e-mail лицензии**.

## Установка за 5 команд

```bash
# 1. Подключиться к серверу (с вашего компьютера)
ssh root@IP_СЕРВЕРА

# 2. Загрузить установщик (с вашего компьютера, в отдельном окне)
scp install.sh root@IP_СЕРВЕРА:/root/

# 3. Сделать скрипт исполняемым (на сервере)
chmod +x /root/install.sh

# 4. Запустить установку (на сервере)
sudo /root/install.sh

# 5. Ответить на вопросы мастера и дождаться сообщения «Установка завершена»
```

## Что спросит мастер
Домен → лицензионный ключ → e-mail лицензии → e-mail и пароль админа →
HTTPS (`y`/`N`) → SMTP / Etherscan / AMLBot *(можно пропустить через Enter)*.

## После установки
- Сайт: `https://ВАШ_ДОМЕН`
- Админка: `https://ВАШ_ДОМЕН/admin/login`
- Данные входа: в `/opt/exchangekit/.credentials` → сохраните и удалите файл.

## Частые команды
```bash
systemctl restart exchangekit-api        # перезапустить
journalctl -u exchangekit-api -f         # логи
certbot --nginx -d ВАШ_ДОМЕН             # включить HTTPS позже
bash /opt/exchangekit/INSTALL/update.sh  # обновить
```
